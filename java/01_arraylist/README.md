# Find Max and Min in ArrayList
A simple Java program that reads integers from the user, stores them in an ArrayList, and finds the smallest and largest values.

## Description
This exercise was built with the goal of practicing the use of the `ArrayList` data structure in Java and reinforcing fundamental programming logic.

The program reads a sequence of numbers from the user, stores them in a dynamic list, and then iterates through the list to determine the minimum and maximum values.

The exercise also reinforces concepts such as variable initialization, loops, comparisons, and algorithmic thinking.

## Concepts practiced
- Java `ArrayList`
- Loops (`for`)
- Conditional statements (`if`)
- Basic algorithm design
