# Count Numbers and Calculate Average exercise
A simple Java program that reads integers until the user types 0, then displays how many numbers were entered and their average — with the list sorted in ascending order.

## Description
This exercise was built with the goal of practicing programming logic without relying on built-in methods like Collections.sort() or any other Java utility. The main focuses of this exercise were:

* Practicing the do-while loop, which fits naturally when you need to read input at least once before checking a stop condition
* Working with the ArrayList data structure to dynamically store an unknown number of elements
* Implementing Bubble Sort manually to sort the list in ascending order
* Reinforcing programming logic by solving the problem from scratch
