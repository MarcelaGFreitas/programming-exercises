Academic Management System (C)
A simple C program developed to review fundamental programming concepts by processing student data and determining their academic status.

## Description
This exercise focuses on reinforcing core concepts of the C programming language:

- User input handling with `scanf`
- Use of arrays to store student status
- Conditional statements (`if`, `else if`, `else`)
- Loop structures (`for`, `do-while`)
- Basic data validation (attendance between 0 and 100)
- Calculation of averages

The program allows the user to input grades and attendance for multiple students and then displays their academic status (Approved, Recovery, or Failed).
